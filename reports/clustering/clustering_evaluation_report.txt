ИТОГОВЫЙ ОТЧЕТ ПО ОЦЕНКЕ МЕТОДОВ КЛАСТЕРИЗАЦИИ
============================================================

ЦЕЛЬ ИССЛЕДОВАНИЯ:
Тематическая кластеризация отзывов клиентов банка для выявления
основных групп обсуждаемых продуктов и услуг.

ИСПОЛЬЗОВАННЫЕ МЕТОДЫ:
1. Кластеризация на основе эмбеддингов (BERT/sentence-transformers)
2. Тематическое моделирование (LDA/BERTopic)
3. Кластеризация на основе TF-IDF (K-means, Agglomerative, DBSCAN)

ВЫВОДЫ И РЕКОМЕНДАЦИИ:
------------------------------
1. Для практического применения рекомендуется использовать метод с наивысшим рейтингом.
2. При выборе метода учитывайте баланс между чистотой кластеров и количеством outliers.
3. Для интерпретации результатов оптимально использовать 10-20 кластеров.
4. Рекомендуется дополнительно проанализировать outliers для выявления новых паттернов.

ПРИМЕНЕНИЕ РЕЗУЛЬТАТОВ:
-------------------------
- Автоматическая категоризация новых отзывов
- Мониторинг настроений по конкретным продуктам
- Выявление проблемных областей в банковском обслуживании
- Персонализация коммуникации с клиентами
- Улучшение продуктовой линейки на основе обратной связи
